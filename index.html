Steganography with Images

This program takes in an image and string of text, and secretely embeds the text into the image. The embedText algorithm manipulates the least significant bit of the RGB value of each pixel, so shorter text embedding is relatively imperceptible. For longer text documents, the program will continue to change the next least significant digit.
